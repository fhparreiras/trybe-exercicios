Fatos sobre o Stomatopoda

Informações Científicas

O Stomatopoda(ou estomatópode), popularmente conhecido no Brasil como tamarutaca ou lacraia-do-mar, tem o nome científico Odontodactylus Scyllarus.
Classificação Científica
Reino: Animalia
Filo: Arthropoda
Subfilo: Crustacea
Classe: Malacostraca
Subclasse: Hoplocarida
Ordem: Stomatopoda

A visão mais complexa do mundo

O Stomatopoda possui o mais complexo sistema de visão de cores do mundo animal. Enquanto nós seres humanos possuímos apenas três receptores, que correspondem à luz azul, verde e vermelha, esse animal possui doze. Portanto, o espectro de cores que os Stomatopodas enxergam, é inimaginável para o cérebro humano.

Feito para matar

Este animal pode parecer glorioso e pacífico, mas é um caçador nato. Seu segundo par de patas, que lembra o formato das patas de um louva-a-Deus, serve para defesa e, principalmente para o ataque. A velocidade de movimento que o Stomatopoda consegue colocar nessas patas é tão grande que se o humano conseguisse fazer um décimo disso para arremessar uma bola de baseball por exemplo, ele a colocaria em órbita. As garras se movem tão rapidamente, que a água ao redor ferve, e é capaz de matar a presa mesmo errando o alvo.
